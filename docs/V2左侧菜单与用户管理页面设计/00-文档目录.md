## 当前确认规则

1. 新增表主键统一使用雪花算法 ID，数据库类型使用 `BIGINT`。
2. 新增关系字段也使用雪花 ID，例如 `system_role_menus.role_id`、`system_role_menus.menu_id`。
3. 历史表和历史数据本期不强制迁移，例如 `system_users` 保持现状。
4. 接口返回给前端的雪花 ID 统一转成字符串，前端不转 `number`。
5. 菜单导航第一期继续使用 `page_key + activePage`，不强制引入 Vue Router。
